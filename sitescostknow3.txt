COPILOT STUDIO TOOL DESCRIPTION
Paste into: Tools > Power BI "Run a query against a dataset" > Description field


Queries live data from the Halliburton Sites and Costs Summary Power BI semantic model. Use this tool whenever the user asks about site counts, NBV, ROU, expired contracts, billing, costs, facilities, contracts, PSL coverage, or risk indicators.

TABLES AND KEY COLUMNS:

CurMonData table - main data table with all site details and expense measures:
  Key columns: [Site and Name], [City], [Country Name], [State], [Street], [Region], [RE Region], [RE Region / MPL], Hemisphere, [Active/Terminated], [Commercial / Housing], [Main usage type], [Property type], [Facility Name], [PSL Name], [Arch.Obj. ID], [Name of Architectural Object], [Term End], [Expired Contacts], [Expired Contracts w/No SR], [Active Contracts on Terminated Sites], [Active Leased Sites w/No Contracts], [Active/Unique Sites?], [Contract No and Name], [Postal Code]
  Key measures: [Current Year NBV], [Current Year ROU], [Measurement - FT2], [Measurement - M2], [Land Measurement - SF], [Allocated FT2 Total], [Wireline and Perforating], [Cementing], [Support], [Testing and Subsea], [Baroid], [Halliburton Drill Bits & Services], [Halliburton Project Management], [Landmark Software & Services], [Completion Tools], [Facility Cost - Current Year], [Rental Expense - Current Year], [Janitorial - Current Year], [Depreciation - Current Year], [Insurance - Current Year], [Total Current Year Expenses YTD], [Total Assets], [ROU Asset Value]

Billing File table - cost allocation:
  Columns: Region, Country, [Facility Name], [PSL Name], [PSL CC], Site, [Sub PSL Name]
  Measures: [Allocated USD], [Billable FT2], [% of Site Cost]

Need Attention table - facilities needing review:
  Columns: [Active/Terminated], [Arch.Obj. ID], [Area Name (From CC)], City, [Commercial / Housing], [Country Name], Hemisphere, [Name of Architectural Object], [RE Region], [RE Region / MPL], Region, Street

CurMonNBV table - NBV by asset:
  Columns: [Arch.Obj. ID], [Name of Architectural Object], Street, City, State, Category, Class, [Cost Ctr], [RE Object], [Re Region/MPL], Site, Status
  Measures: [Current Year NBV], [Next Year NBV], [Next 2 Years NBV], [Next 3 Years NBV], [Next 4 Years NBV], [Total Assets], [Total Acq Value], [Land Acq Value], [Non Land Acq Value]

REISPO table - space and occupancy:
  Columns: [PSL Name], [Sub PSL Name], Site, Occupant, [MPL Site Name], [Unit of Capacity], [M2 or FT2]
  Measures: [FT2], [M2], [Capacity], [Capacity in SF], [Pct]

KeyMeasures table - calculated KPI measures:
  [Count_Expired_Contract] = expired contracts count
  [Count_Expired_Contract_No_SR] = expired with no service request
  [Count_ActContr_on_TermSite] = active contracts on terminated sites
  [Count_ActLeasedSite_No_Contract] = leased sites without contracts
  [Count_Contract] = total contracts
  [Count_Presentation] = presentation site count
  [Count_Site] = total site count
  [Count_Impaired] = impaired count
  [Value_NBV] = total Net Book Value
  [Value_ROU] = total Right of Use
  [Value_Building] = total building FT2
  [Value_Land] = total land FT2
  [NBV_CY] = current year NBV
  [NBV_TotalAsset] = total asset NBV

ZFAF table - asset financials:
  Columns: Region, Country, [Cost Center], [Company code], [Project Object], Description
  Measures: [Asset Value], [Budget], [WBS Value], [WBS Commitments]

EXAMPLE DAX QUERIES:

Overview KPIs:
EVALUATE ROW("Expired Contracts", [Count_Expired_Contract], "Expired No SR", [Count_Expired_Contract_No_SR], "Active on Terminated", [Count_ActContr_on_TermSite], "Leased No Contract", [Count_ActLeasedSite_No_Contract], "Total Contracts", [Count_Contract], "Sites", [Count_Site], "NBV", [Value_NBV], "ROU", [Value_ROU], "Building FT2", [Value_Building], "Land FT2", [Value_Land])

Billing by country:
EVALUATE CALCULATETABLE(SUMMARIZECOLUMNS('Billing File'[Facility Name], 'Billing File'[PSL Name], "Cost", SUM('Billing File'[Allocated USD]), "FT2", SUM('Billing File'[Billable FT2])), 'Billing File'[Country] = "Australia")

All billing data:
EVALUATE SUMMARIZECOLUMNS('Billing File'[Region], 'Billing File'[Country], 'Billing File'[Facility Name], 'Billing File'[PSL Name], "Cost", SUM('Billing File'[Allocated USD]), "FT2", SUM('Billing File'[Billable FT2]))

Facilities needing attention:
EVALUATE SUMMARIZECOLUMNS('Need Attention'[Arch.Obj. ID], 'Need Attention'[Name of Architectural Object], 'Need Attention'[City], 'Need Attention'[Street], 'Need Attention'[Region], 'Need Attention'[Active/Terminated], 'Need Attention'[Commercial / Housing])

Facilities in a specific city:
EVALUATE FILTER('Need Attention', 'Need Attention'[City] = "Comodoro Rivadavia")

NBV by asset:
EVALUATE SUMMARIZECOLUMNS('CurMonNBV'[Arch.Obj. ID], 'CurMonNBV'[Name of Architectural Object], 'CurMonNBV'[City], 'CurMonNBV'[Street], 'CurMonNBV'[Status], "NBV", SUM('CurMonNBV'[Current Year NBV]))

Site details by country:
EVALUATE CALCULATETABLE(SUMMARIZECOLUMNS('CurMonData'[Site and Name], 'CurMonData'[City], 'CurMonData'[Active/Terminated], 'CurMonData'[Commercial / Housing], 'CurMonData'[Main usage type]), 'CurMonData'[Country Name] = "Australia")

PSL occupancy:
EVALUATE SUMMARIZECOLUMNS('REISPO'[PSL Name], 'REISPO'[Site], 'REISPO'[Occupant], "FT2", SUM('REISPO'[FT2]))

Expense breakdown for a site:
EVALUATE CALCULATETABLE(ROW("Facility Cost", [Facility Cost - Current Year], "Rental", [Rental Expense - Current Year], "Janitorial", [Janitorial - Current Year], "Depreciation", [Depreciation - Current Year], "Total Expenses", [Total Current Year Expenses YTD]), 'CurMonData'[Country Name] = "Australia")
