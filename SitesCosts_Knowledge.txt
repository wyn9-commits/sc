## Report Overview

The Sites and Costs Summary is a Power BI report used by Halliburton's Real Estate and Facilities Management team. It tracks global property assets, lease contracts, facility costs, and Net Book Values across all Halliburton operating regions.

The report is located at: Power BI workspace ID df57b8cd-aa06-4892-91c3-a560be5031a8

## Common Abbreviations

- NBV: Net Book Value — the book value of a physical asset in USD
- ROU: Right of Use — the value of a lease obligation in USD
- FT2: Square footage — building or land area measurement
- PSL: Product Service Line — Halliburton's business unit categorization
- PSL CC: PSL Cost Center — the cost center code for each PSL
- Sub PSL: Sub-division within a Product Service Line
- HCT: Halliburton Completion Tools (a PSL)
- W&P: Wireline and Perforating (a PSL)
- SR: Service Request
- Arch.Obj. ID: Architectural Object ID — unique identifier for a physical building or asset
- RE Region: Real Estate Region
- MPL: Market Planning Level
- CMP: Ciudad del Carmen (region code)
- CHU: Chubut province, Argentina (region code)
- NEU: Neuquen province, Argentina (region code)
- ANZ: Anzoátegui state, Venezuela (region code)

## Top 10 Question Patterns

These are the most frequently asked questions by the facilities management team:

### 1. Active Housing/Commercial facilities in a location
Example: "What are the active Housing/Commercial facilities in X City/Country/Area/Region?"
Answer approach: Filter by Active/Terminated = Active, then by Commercial/Housing type, then by the specified location. Return facility names, cities, and site IDs.

### 2. Net Book Value of a location
Example: "What is the Net Book Value of X Facility/Country/Area/Region?"
Answer approach: Sum the NBV measure filtered by the specified location. Format as USD with commas. Refer to the NBV by Facility page.

### 3. Asset list or register for a location
Example: "Provide an asset list/register for X Facility/Country/Area/Region."
Answer approach: List all architectural objects (Arch.Obj. ID, Name of Architectural Object, Street, City, State, Postal Code) filtered by the specified location. Refer to the NBV by Facility page.

### 4. ROU value of a location
Example: "What is the ROU value of X Facility/Country/Area/Region?"
Answer approach: Return the ROU measure filtered by the specified location. Format as USD with commas. Refer to the Overview page.

### 5. Active contracts for a location
Example: "What are the active contracts for X Facility/Country/Area/Region?"
Answer approach: List Contract # and Name, Contract Category, and Term End date, filtered by Active status and the specified location. Refer to the Active Facilities by Region page.

### 6. Facilities occupied by a PSL
Example: "What facilities do Cementing/Wireline/HCT/etc. occupy?"
Answer approach: Filter by PSL Name and return facility names, regions, and countries along with the Facilities Count. Refer to the Active Facilities by Region page.

### 7. Facilities by building type
Example: "Provide a list of Warehouse/Plant/Manufacturing/etc. facilities globally or in X Facility/Country/Area/Region."
Answer approach: Filter by the building/usage type (Space Type or Main Usage Type) and optionally by location. Return facility names with full location details.

### 8. Operating expense for a time period and location
Example: "What is operating expense for X period of time for Facility/Country/Area/Region?"
Answer approach: Return Sum of Allocated USD filtered by the specified location and time period. Include Sum of Billable FT2 for context. Refer to the Actual Rate Billing page.

### 9. Specific expense category for a location
Example: "What is the rental/depreciation/janitorial/etc. expense for X Facility/Country/Area/Region?"
Answer approach: Filter by the expense/cost category and location, then return the allocated USD amounts. Refer to the Actual Rate Billing page.

### 10. Vacant facilities
Example: "What facilities are vacant for X space type?"
Answer approach: Filter by the Space Type and vacant/unoccupied status. Return facility names and location details.

### Additional patterns
- "Provide a list of Owned/Leased/Customer Owned facilities globally or in X location" — filter by ownership/contract type and location.
- "Provide a list of active housing/commercial facilities in X location for a specific time period in the past" — filter by type, location, and time period.

## Risk Indicators

When data shows any of the following, flag them clearly as risks:

1. Expired Contracts (currently 124 / 199 / 98 across categories): These are contracts past their term end date. High count indicates a renewal backlog that may cause compliance issues.

2. Active Contracts on Terminated Sites (currently 5 / 1 / 10): These are contracts still active on sites that have been marked as terminated. This is a compliance and financial risk — the organization may be paying for sites no longer in use.

3. Active Leased Sites - No Contracts (currently 10 / 6 / 2): These are sites that are actively leased but have no associated contract in the system. This is a governance gap — the organization has lease obligations without documented agreements.

4. Expired Contracts no SR (currently 6 / 2 / 1): Expired contracts without a service request filed to renew or terminate them. These need immediate attention.
